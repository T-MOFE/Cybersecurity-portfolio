## 📊 Methodology

Every engagement in this portfolio follows the standard penetration testing lifecycle:

```
1. Reconnaissance    →  Passive & active information gathering
2. Enumeration       →  Service discovery, version detection
3. Exploitation      →  Targeted attack based on findings
4. Post-Exploitation →  Privilege escalation, lateral movement
5. Reporting         →  CVSS scoring, remediation guidance
```
